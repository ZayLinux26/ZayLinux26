## 🎯 Why This Combination?

These three tracks are not separate hobbies. They are the same security problem viewed from three layers of the enterprise stack.

```
┌─────────────────────────────────────────────────────────────────┐
│                    ENTERPRISE SECURITY FLOW                      │
│                                                                  │
│  👤 User / Admin                                                 │
│       │                                                          │
│       ▼                                                          │
│  ┌─────────────────┐    Authenticate    ┌──────────────────┐    │
│  │   Entra ID      │◄──────────────────►│    CyberArk      │    │
│  │   Identity      │    MFA + SSO       │    Vault         │    │
│  └─────────────────┘                    └──────────────────┘    │
│       │                                        │                 │
│       │ Cloud Resources                        │ Privileged      │
│       ▼                                        ▼ Credentials     │
│  ┌─────────────────┐                    ┌──────────────────┐    │
│  │  Azure VMs &    │◄───────────────────│   PSM Session    │    │
│  │  Services       │    Secure Access   │   Recording      │    │
│  └─────────────────┘                    └──────────────────┘    │
│       │                                        │                 │
│       ▼                                        ▼                 │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │              RHEL Servers (Backend Infrastructure)       │    │
│  │         • Database Servers    • Application Servers      │    │
│  │         • Web Servers         • Domain Controllers       │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

Identity is the perimeter. PAM is the safe inside. Linux is the floor the safe sits on. If you can't operate all three together, you can't actually secure the enterprise.
